# Spring Boot Post & Comment API

A RESTful API built with Spring Boot for managing **posts** and **comments** with **user authentication**.  
Users can register, log in, create posts, and comment on posts.  

---

## Features

- User registration and login with **password hashing** (BCrypt)
- CRUD operations for Posts
- Add comments to posts
- DTOs for clean request/response handling
- Exception handling with `ResponseEntity`
- MySQL database integration

---

## Technologies

- **Java 17**  
- **Spring Boot** (Web, Data JPA)  
- **MySQL**  
- **Lombok**  
- **Postman** (for testing API endpoints)
